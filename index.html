Notable Projects

Phishing Email Analyzer | Python

Developed a Python-based tool to analyze .eml email headers and detect phishing indicators. The program parses email metadata including sender details, subject lines, and Received headers to identify suspicious patterns commonly used in phishing campaigns. The tool extracts sender IP addresses and highlights suspicious keywords associated with malicious emails. Automated analysis output helps investigators quickly identify potentially malicious emails. This project demonstrates practical email forensics and phishing investigation techniques used in cybersecurity incident analysis.

GitHub: Phishing Email Analyzer Repository

SOC Log Analyzer | Python

Developed a security log analysis tool designed to detect suspicious authentication activity from system logs. The tool automatically parses log files to identify repeated failed SSH login attempts and unusual authentication patterns. Detection logic was implemented to flag potential brute-force attacks based on repeated login failures from specific IP addresses. The tool generates SOC-style alerts highlighting suspicious activity and potentially malicious IP addresses. This project demonstrates practical skills in log analysis, threat detection, and security monitoring used in Security Operations Centers (SOC).

GitHub: SOC Log Analyzer Repository

Threat Intelligence IP Reputation Checker | Python

Developed a threat intelligence tool that extracts IP addresses from network logs and evaluates their reputation using external threat intelligence sources. The tool integrates the AbuseIPDB API to retrieve abuse confidence scores and historical reports associated with IP addresses. Automated risk classification helps identify potentially malicious IPs and prioritize investigation efforts. This project demonstrates a practical workflow combining log analysis, IOC enrichment, and automated threat investigation techniques commonly used in security operations and threat intelligence analysis.

GitHub: Threat Intelligence IP Checker Repository

Financial Fraud Detection System

Developed a hybrid machine learning system designed to identify and prevent fraudulent financial transactions. The project combined multiple machine learning algorithms including Neural Networks, Random Forest, and XGBoost to improve detection accuracy and reduce false positives. A dataset containing transaction records was analyzed to identify suspicious behavioral patterns such as unusual transaction frequency, abnormal transaction amounts, and irregular geographic activity. The system involved data preprocessing, feature engineering, and model evaluation to optimize fraud detection performance. By leveraging ensemble learning techniques, the solution was able to capture complex patterns associated with fraudulent activity. This project strengthened my understanding of anomaly detection, data analysis, and the application of machine learning models in cybersecurity and financial security contexts.

Cybersecurity Monitoring Lab

Designed and implemented a cybersecurity monitoring lab to simulate real-world security operations and incident investigation scenarios. The lab environment was used to practice log analysis, system monitoring, and network security troubleshooting. Various system and authentication logs were examined to identify suspicious activity, unauthorized access attempts, and potential security incidents. Within this environment, I configured firewall rules, analyzed authentication failures, and investigated abnormal system behaviors to understand how attackers attempt to exploit vulnerabilities. The lab also helped develop practical skills in security monitoring, incident response procedures, and troubleshooting security configurations. This project enhanced my ability to interpret security logs, investigate alerts, and strengthen defensive security practices within network environments.
